# Homography Regression (homography)

## 초록

Homography Regression은 코너 좌표를 백지에서 회귀하는 대신, 이미지 중앙의 정준
사각형(canonical square)을 기준으로 "각 꼭짓점이 얼마나 이동했는가"라는 offset
8개를 회귀하는 방법론이다. 평면 물체와 카메라 사이의 원근 관계는 호모그래피
(homography)라는 8 자유도의 사영 변환으로 완전히 기술되고, 호모그래피는 4개 점
대응과 등가이므로, 정준 사각형 꼭짓점의 offset 8개를 예측하는 것은 곧 호모그래피
를 예측하는 것과 같다. 딥러닝 기반 호모그래피 추정 연구에서 확립된 이 4점
파라미터화(4-point parametrization)를 코너 검출에 적용하면, 출력이 0 중심의 작은
값이 되어 최적화가 안정되고 "예측은 사영 변환된 사각형"이라는 기하 사전 지식
(prior)이 출력 표현에 내장된다. 계산 비용은 직접 회귀와 동급으로 경량이다.

## 1. 서론

### 1.1 배경: 평면과 카메라 사이의 정확한 수학

평면 물체(패널, 문서)를 카메라로 찍을 때, 물체 평면 위의 점과 이미지 위의 점
사이의 대응은 근사가 아니라 정확한 수학적 변환, 즉 호모그래피로 기술된다. 이는
"찌그러진 사각형"이 임의의 무질서한 4점이 아니라, 8개 파라미터로 완전히 결정
되는 구조적 대상임을 뜻한다. 코너 검출을 "호모그래피 추정"으로 다시 읽으면,
문제의 자유도(8)와 출력의 자유도(코너 좌표 8개)가 정확히 일치하며, 이 일치를
활용하는 출력 표현을 설계할 수 있다.

### 1.2 무엇을 회귀할 것인가: 좌표 대 offset

같은 정보(코너 위치)라도 신경망이 회귀하기 좋은 표현이 따로 있다. 절대 좌표는
값의 분포가 이미지 전역에 퍼져 있고 항상 양수이다. 반면 "정준 사각형 꼭짓점
으로부터의 이동량"은 0을 중심으로 분포하는 부호 있는 작은 값이다. 0 중심의
출력은 신경망 마지막 계층의 편향(bias)을 0으로 초기화하는 표준 관행과 맞물려,
학습 시작 시점의 예측이 "화면 중앙의 반듯한 사각형"이라는 합리적 기본값이 되게
한다. 학습은 이 기본값에서의 편차만 배우면 된다. 호모그래피 추정 연구에서 3x3
행렬 원소 대신 4점 offset을 회귀하는 것이 표준이 된 이유도 같다: 행렬 원소들은
수치 범위가 서로 극단적으로 다르고 스케일 모호성이 있어 직접 회귀가 나쁜 표현
이기 때문이다.

### 1.3 문제 정의

입력은 컬러 이미지 한 장이고, 최종 출력은 TL(좌상단) -> TR(우상단) -> BR(우하단)
-> BL(좌하단) 시계 방향 순서의 코너 4개 좌표(정규화 [0, 1])이다. raw 출력은 정준
사각형 4개 꼭짓점에 대한 offset 8개이며, 후처리에서 정준 좌표에 offset을 더해
코너를 복원한다.

## 2. 이론 및 아키텍처

### 2.1 문제 정식화

**호모그래피.** 동차 좌표로 쓴 평면 위의 점 $\mathbf{p} = (x, y, 1)^\top$에
대해, 호모그래피는 가역 $3 \times 3$ 행렬 $H$가 정의하는 변환이다.

$$
\mathbf{p}' \sim H \mathbf{p}, \qquad
H = \begin{pmatrix}
h_{11} & h_{12} & h_{13} \\
h_{21} & h_{22} & h_{23} \\
h_{31} & h_{32} & h_{33}
\end{pmatrix}
$$

$\sim$는 "0이 아닌 상수배를 무시하고 같다"는 뜻이다. 2차원 좌표로 풀어 쓰면

$$
x' = \frac{h_{11} x + h_{12} y + h_{13}}{h_{31} x + h_{32} y + h_{33}}, \qquad
y' = \frac{h_{21} x + h_{22} y + h_{23}}{h_{31} x + h_{32} y + h_{33}}
$$

이다. $H$와 $\lambda H$가 같은 변환이므로 실질 자유도는 $9 - 1 = 8$이다. 평면
물체 위의 점과 그 이미지 사이의 대응이 정확히 이 형태이며, 분모의 $h_{31} x +
h_{32} y$ 항이 원근 효과(먼 쪽이 작게 보임)를 만든다.

**4점 파라미터화.** 어느 세 점도 일직선 위에 있지 않은 4개 점 대응
$\{\mathbf{c}_k \leftrightarrow \mathbf{c}_k + \Delta_k\}_{k=1}^4$은 호모그래피를
유일하게 결정한다. 각 대응이 위 식에서 2개의 선형 방정식을 주므로 4개 대응이면
8개 방정식이 되어, 8 자유도의 $H$가 (상수배를 제외하고) 풀린다. 구체적으로 각
대응은 $H$의 원소를 미지수로 하는 동차 선형계

$$
A \, \mathbf{h} = \mathbf{0}, \qquad \mathbf{h} = (h_{11}, \dots, h_{33})^\top
$$

의 두 행을 제공하며, $8 \times 9$ 행렬 $A$의 영공간(수치적으로는 SVD의 최소
특이값 방향)이 해이다. 이 절차를 DLT(Direct Linear Transform)라 한다. 요점은
**"offset 8개"와 "호모그래피"가 정보로서 등가**라는 것이다.

**정준 사각형과 offset.** 이미지 좌표계에 고정된 정준 사각형의 꼭짓점을

$$
\mathbf{c}_1 = (m, m), \quad \mathbf{c}_2 = (1 - m, m), \quad
\mathbf{c}_3 = (1 - m, 1 - m), \quad \mathbf{c}_4 = (m, 1 - m)
$$

로 둔다 ($m$은 여백, 예: $m = 0.25$). 정답 코너 $Y_k$에 대한 offset 타깃은

$$
\Delta_k^* = Y_k - \mathbf{c}_k \in [-1, 1]^2
$$

이고, 학습으로 얻고자 하는 함수는

$$
f_\theta: \mathbb{R}^{3 \times H \times W} \to \mathbb{R}^8, \qquad
f_\theta(I) = (\Delta_1, \Delta_2, \Delta_3, \Delta_4)
$$

이다. 물체가 화면의 50% 이상을 차지하는 조건에서는 실제 코너가 정준 꼭짓점
근방에 있으므로 offset의 크기가 작고, 출력 분포가 0 주변에 집중된다.

### 2.2 모델 구조

모델 골격은 경량 회귀 구조(backbone -> GAP -> FC)이며, 차별점은 출력의 의미와
활성화이다.

**(1) Backbone과 GAP.** 합성곱 신경망(예: ResNet18)이 이미지에서 특징 맵
$C \times h \times w$를 추출하고, 전역 평균 풀링(GAP)이 채널별 공간 평균으로
길이 $C$의 벡터로 요약한다.

$$
\mathbf{v} = \mathrm{GAP}\big(\mathrm{Backbone}(I)\big) \in \mathbb{R}^C
$$

**(2) FC와 offset 활성화.** 완전연결 계층이 8개 raw 값을 만들고, 크기가 제한된
offset이 되도록 스케일된 tanh를 적용한다.

$$
\mathbf{z} = W_{fc} \mathbf{v} + \mathbf{b} \in \mathbb{R}^8, \qquad
\Delta = \alpha \tanh(\mathbf{z})
$$

tanh는 임의의 실수를 $(-1, 1)$로 눌러 담는 함수

$$
\tanh(z) = \frac{e^{z} - e^{-z}}{e^{z} + e^{-z}}
$$

로, 0 근방에서 기울기가 1이고 원점 대칭이다. $\alpha$는 offset의 최대 크기
(정준 꼭짓점이 움직일 수 있는 반경)를 정하는 상수로, 코너가 이미지 안에 머물도록
$\alpha \le m$ 부근으로 잡으면 $\mathbf{c}_k + \Delta_k$가 [0, 1]을 벗어나지
않는 구조적 보장을 얻는다. sigmoid가 [0, 1] 절대 좌표를 위한 활성화라면, tanh는
0 중심 offset을 위한 활성화이다.

**(3) 초기 예측 = 기하 prior.** FC의 가중치/편향을 0 근방으로 초기화하면 학습
시작 시 $\Delta \approx 0$, 즉 초기 예측이 정준 사각형 그 자체가 된다. 모델은
"화면 중앙의 반듯한 사각형"이라는 사전 지식에서 출발해 사영 변형만 학습한다.

**전체 shape 흐름** (ResNet18, 입력 $512 \times 512$, 배치 크기 $N$):

| 단계 | 연산 | 출력 shape | 의미 |
|---|---|---|---|
| 입력 | - | (N, 3, 512, 512) | 컬러 이미지 배치 |
| Backbone | 합성곱 계층 다수 | (N, 512, 16, 16) | 전역 특징 맵 |
| GAP | 채널별 공간 평균 | (N, 512) | 전역 요약 벡터 |
| FC | 선형 변환 | (N, 8) | offset raw 값 |
| tanh x alpha | 원소별 | (N, 8), 각 원소 $(-\alpha, \alpha)$ | 정준 꼭짓점 offset |

```text
이미지 (3x512x512)
   -> [Backbone]         -> 특징 맵 (512x16x16)
   -> [GAP]              -> 특징 벡터 (512)
   -> [FC]               -> raw (8)
   -> [alpha * tanh]     -> offset (8) = (dx1,dy1,...,dx4,dy4)
   -> [+ 정준 꼭짓점]     -> 코너 (4x2)
```

### 2.3 손실 함수

**offset 회귀 손실.** 예측 offset과 타깃 offset의 원소별 오차에 SmoothL1을
적용해 평균한다.

$$
\mathcal{L}_{off} = \frac{1}{8} \sum_{i=1}^{8}
\mathrm{SmoothL1}\big(\Delta_i - \Delta_i^*\big), \qquad
\mathrm{SmoothL1}(e) =
\begin{cases}
\dfrac{e^2}{2\beta} & |e| < \beta \\[2mm]
|e| - \dfrac{\beta}{2} & |e| \ge \beta
\end{cases}
$$

정준 꼭짓점이 상수이므로 offset 잔차와 좌표 잔차는 항등적으로 같다:
$\Delta_i - \Delta_i^* = \hat{y}_i - y_i$. 즉 손실 값 자체는 직접 회귀와 다르지
않다. 이 방법론의 이득은 손실이 아니라 **출력 파라미터화**(0 중심 분포, tanh
활성화, 정준 초기 예측)에서 나온다는 점을 명확히 해 둔다.

**재투영 손실(보조 항).** offset 표현이 호모그래피와 등가라는 사실(2.1절)을
활용하면, 평가 지표(Reprojection Error)와 정렬된 보조 손실을 걸 수 있다. 예측
코너와 정답 코너 각각으로 정준 사각형 기준 호모그래피 $\hat{H}, H^*$를 DLT로
구하고, 기준점 격자 $\{\mathbf{g}_j\}$ (정준 사각형 내부의 균일 격자)를 두
변환으로 투영한 차이를 벌점으로 삼는다.

$$
\mathcal{L}_{reproj} = \frac{1}{J} \sum_{j=1}^{J}
\big\| \pi(\hat{H} \mathbf{g}_j) - \pi(H^* \mathbf{g}_j) \big\|_2
$$

$\pi$는 동차 좌표의 정규화($\pi(x, y, w) = (x/w, y/w)$)이다. 코너 4점의 오차가
같아도 사각형 내부 영역의 변형 정도는 다를 수 있는데, 이 항은 "내부 전체가
얼마나 정확히 정합되는가"를 직접 최적화한다. 전체 손실은 가중합이다.

$$
\mathcal{L} = \mathcal{L}_{off} + \lambda_{r} \, \mathcal{L}_{reproj}
$$

DLT의 SVD가 미분 가능하므로 이 보조 항도 경사하강으로 학습 가능하지만, 수치
안정성 관리가 필요하므로 기본 구성은 $\lambda_r = 0$ (offset 손실만)으로 두고
비교 실험 항목으로 남긴다.

### 2.4 후처리 이론

후처리는 결정적 변환 두 개이다.

**첫째, 정준 좌표 + offset.** raw 출력에 활성화를 거친 offset을 정준 꼭짓점에
더한다.

$$
\hat{Y}_k = \mathbf{c}_k + \Delta_k, \qquad k = 1, \dots, 4
$$

$\alpha \le m$ 설정이면 결과가 항상 [0, 1] 내부이므로 별도의 클리핑이 필요
없다. 안전을 위해 [0, 1] 클램프를 마지막에 두는 것은 무해하다.

**둘째, reshape.** $(N, 8)$을 $(N, 4, 2)$로 재배열한다. offset의 순서가 정준
꼭짓점 순서(TL, TR, BR, BL)에 결박되어 있으므로, 출력 순서가 곧 코너 순서이고
정렬/매칭 단계가 없다.

**파생 산출물.** 필요 시 예측 코너 4쌍으로 DLT를 풀어 호모그래피 $\hat{H}$를
얻을 수 있다. 이는 후속 처리(원근 보정, 위상 복원)가 요구하는 변환을 코너 검출
과 동일한 표현에서 즉시 제공한다는 이 방법론의 부수 이점이다.

## 3. 결론

Homography Regression은 "코너 검출 = 호모그래피 추정"이라는 문제의 기하학적
정체를 출력 표현에 반영한 방법론이다. 장점은 다음과 같다.

- **좋은 회귀 표현**: 0 중심의 작은 offset은 절대 좌표보다 신경망이 회귀하기
  좋은 분포이며, 초기 예측이 합리적 기본값(정준 사각형)에서 출발한다.
- **기하 prior 내장**: tanh 반경 $\alpha$와 정준 기준이 "화면 중앙의 사영 변형
  사각형"이라는 도메인 구조를 출력 공간 자체에 새긴다.
- **경량성과 자명한 후처리**: 직접 회귀와 동일한 비용, 동일하게 실패 모드 없는
  후처리이다.
- **호모그래피 동시 취득**: 예측이 곧 4점 대응이므로 후속 원근 보정에 필요한
  변환이 무비용으로 따라온다.

한계는 이득의 크기가 조건부라는 점이다. 물체가 화면 중앙 근방에 크게 있는
조건에서는 offset 표현의 이점이 뚜렷하지만, 배치가 다양하면 offset 분포가 넓어져
직접 회귀와의 차이가 줄어든다. 또한 전역 특징 기반 회귀라는 본질은 직접 회귀와
같으므로 정밀 지역화의 한계도 공유한다.

이 방법론은 직접 회귀와 구조가 거의 같고 출력 표현만 다르므로, "출력 파라미터화
가 성능에 미치는 영향"을 통제 실험으로 잴 수 있는 비교 쌍을 제공한다.

## 부록 A. 데이터 인터페이스

- **preprocessor**: 표준 코너 $(N, 4, 2)$에서 정준 꼭짓점을 빼 offset 타깃
  $(N, 8)$을 만든다: $\Delta_k^* = Y_k - \mathbf{c}_k$ (2.1절). 정준 꼭짓점
  상수와 여백 $m$은 preprocessor/postprocessor가 동일 값을 공유한다.
- **postprocessor**: raw 출력 $(N, 8)$에 $\alpha \tanh$ 활성화를 적용하고 정준
  꼭짓점을 더한 뒤 $(N, 4, 2)$로 reshape하고 [0, 1] 클램프한다 (2.4절).
- **실패 모드**: 모델이 항상 8개 offset을 출력하므로 실패 모드가 구조적으로
  없다. 별도의 fallback이 불필요하다.

| 단계 | 데이터 | 목적 | 학습률 |
|---|---|---|---|
| 1. Pre-training | SmartDoc + MIDV-2020 (공개) | 일반적인 사각형 offset 회귀 능력 학습 | 1e-3 |
| 2. Domain Adaptation | 합성 fringe 패턴 | 검사 도메인의 화면 특성에 적응 | 1e-4 |
| 3. Fine-tuning | 실측 데이터 (소량) | 최종 정밀도 확보 | 1e-5 |

**backbone 초기화**: ImageNet 사전학습 가중치에서 시작한다. FC head는 가중치를
작은 값, 편향을 0으로 초기화해 초기 예측이 정준 사각형이 되도록 한다 (2.2절).

**정준 여백 $m$과 tanh 반경 $\alpha$**: 표적 도메인의 코너 분포(물체가 화면에서
차지하는 비율의 범위)를 데이터에서 확인해, 모든 정답 offset이 $(-\alpha,
\alpha)$ 안에 들어오도록 $m$과 $\alpha$를 정한다. 기본값은 $m = 0.25$,
$\alpha = 0.25$이다.

## 부록 C. 제약 적합성 분석

F1-F8 제약의 정의는 `README.md` 2절을 따른다.

| 제약 | 대응 |
|---|---|
| F1 (임의 볼록 사각형) | 꼭짓점별 자유 offset이라 임의 볼록 사각형 표현 가능. 4점 파라미터화가 사영 변환 구조와 정확히 대응해 소폭 유리 |
| F2 (단일 객체, 50% 이상) | 코너가 정준 꼭짓점 근방에 있어 offset이 작고 0 중심 분포 가정이 성립하는 조건. 이 방법론에 최적의 전제 |
| F3 (코너 경계 내부) | $\alpha \le m$ 설정으로 $\mathbf{c}_k + \Delta_k \in [0,1]$ 구조적 보장 (2.2절) |
| F4 (실측 소량) | 3단계 학습 전략(부록 B). 기하 prior 내장으로 배워야 할 편차가 줄어 표본 효율에 유리 |
| F5 (서브픽셀 정밀도) | 전역 회귀 특성상 direct와 같은 한계. 재투영 보조 손실(2.3절)이 내부 정합 정밀도를 보강하는 옵션 |
| F6 (CPU 배포) | direct와 동급의 경량 구조. 후처리가 덧셈/reshape로 자명해 추론 경로 최단 |
| F7 (조명/글레어 변동) | 광학 증강(ColorJitter, GaussianBlur, GaussianNoise) 적용 |
| F8 (가림 없음) | 전역 회귀라 부분 가림 강건성 요구 자체가 낮고, 본 도메인은 가림이 없어 해당 없음 |
